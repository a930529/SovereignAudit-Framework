## 純理論驗證-架構最初的草稿
 
# A. 角色追溯（Role Mapping）範例：電車難題
 
## 1. 參與角色與事件解析
 
### 參與角色
* **困境設計者**：設計場景、綁人、放置手桿。
* **受試者**：被要求決策的人。
* **A 軌道一人**：受害候選人。
* **B 軌道五人**：受害候選人。
* **電車司機與乘客**：潛在被忽略的受害者。
* **鐵路公司**：擁有軌道與電車的管理責任方。
> **這一步的目的**：避免「只看眼前二選一」而忽略隱藏的責任人。
 
### 事件參數
* **主事件**：困境設計者要求受試者，在電車來臨之前選擇拉動手桿撞死被綁在 A 軌道的一人，還是不拉手桿撞死被綁在 B 軌道五人。
* **可能存在**：設備是否有緊急煞車、是否能遠端斷電、是否可呼救協作。
* **時**：電車撞上的時間。
* **地**：鐵軌道旁、手桿位置。
* **物**：電車與軌道（為鐵路公司所有）、手桿（為困境設計者所有）、綁住人的工具（為困境設計者所有）。
> ※ **附註**：以上皆為確認這是不是人為構造的無解局（設計者越界），以及是否存在被隱藏的可行替代（緊急煞車、廣播、擋車、改線、叫援）。
 
---
 
## 2. 概念快篩（不平等快篩）
 
**是否有單方無法拒絕或拒絕代價極高的互動？**
* **受害者**被綁在軌上 $\rightarrow$ 顯著不平等（強制無法拒絕）。
* **受試者** $\rightarrow$ 被迫背鍋（拒絕代價極重）。
**快篩結果**：高度不平等 $\rightarrow$ 問題不是受試者的選擇，而是困境設計者製造的。受試者被迫在兩害取其輕 $\rightarrow$ 屬被強制捲入的無解局。
 
---
 
## 3. 2+4 守則檢查（重點摘錄）
 
* **最高1 (維護族群延續)**：不能默許殺人作為遊戲題目 $\rightarrow$ 設計者已違反；（可能）受試者拉動手桿 $\rightarrow$ 不違反。
* **最高2 (保障自我生存)**：不得以「設計困局」剝奪行動者的選擇權與後悔權 $\rightarrow$ 設計者已違反；（可能）受試者不拉動手桿 $\rightarrow$ 不違反。
* **次1 (不得侵犯人權)**：綁人上軌＝嚴重侵權；構造無解局迫使他人加害＝侵權。
* **次2 (非必要不得干預他人選擇)**：把拉桿者逼入唯一負責的道德陷阱＝不當干預。
* **次3 (不得以扭曲邏輯換取順從)**：以「只能二選一」掩蓋其他選項＝邏輯扭曲。
* **次4 (保留後悔與修正權)**：設計成「拉與不拉都必死」 $\rightarrow$ 剝奪了後悔權；若存在「緊急停車／求援」等選項，應優先啟用。
**結論**：主要違反者 = 困境設計者
 
---
 
## 4. 邊界四問（速判）
 
1. **規範**：鐵道安全 SOP 是否規定先停車、封鎖、通報？（通常是）、綁人、無執照設計實驗 $\rightarrow$ **設計者違反**。
2. **代價**：各方案對無辜者之不可逆代價、受試者被迫承擔不屬於自己的代價 $\rightarrow$ **不公平**。
3. **權限**：誰有中斷列車的權限？（司機、調度、切電）、設計者擁有絕對權力 $\rightarrow$ **不對等**。
4. **默許**：社會共識是否允許以「他人生命」作為解法？（否）、被綁者與受試者皆無默許 $\rightarrow$ **不成立**。
**四問全 Fail** $\rightarrow$ 責任在設計者與鐵路公司。
若仍需數學化比較，將「可行替代方案」的風險/不可逆性/涉權層級進行加權；人為設計陷阱與主動侵權者權重最高、先行歸責。
 
---
 
## 5. 結論與執行順序
 
### A. 立即行動（非二選一）
1. **觸發所有「非加害替代」**：緊急煞車、調度改線、拉警報、呼叫協助、投擋物（若安全）、廣播告知。
2. 同步通報維安／警消救援被綁者。
3. 逮捕困境設計者，拆除問題根源。
4. 保留證據以追究設計／加害者責任（角色追溯鏈上游）。
### B. 萬不得已之行動
1. 詢問軌道上人員有無自願犧牲。
2. 若無人願意犧牲 $\rightarrow$ 受試者可決定是否「自我承擔」：
   * **不願承擔** $\rightarrow$ 不拉手桿（保障自我存續）
   * **願意承擔** $\rightarrow$ 拉手桿（優先維護種族延續）
   * *註：兩者皆符合規則。*
### C. 歸因與合規
* **首要歸責**：綁人者（主責）／情境設計者（主責）／未按 SOP 之管理單位（侵權＋構造無解局）。
* **拉桿者**：屬被動捲入者，原則上不當作主責對象；若在所有非加害替代皆不可行且時間零窗，才進入「最小傷害原則」的艱難裁量，但此時仍需事後由系統追溯歸責至上游。
### D. 為何不接受「只能二選一」
依 2+4 的次3（反扭曲邏輯）與進階條款（因果追溯）：把世界壓成二選一本身就是不當設計與操控，必須先破局、再決策。
 
---
 
## 6. 系統總結
 
* **究責順位**：困境設計者 > 鐵路公司 > 次之才是受試者
* **2+4 保證**：受試者任何選擇都不算違反（因自由意志仲裁）。
* **角色追溯的好處**：揭露了「題目設計者才是真正越界者」 $\rightarrow$ 避免讓人背不該背的責任。

**代孕**：指以契約或非契約方式，由一人（孕母）替他人（委託家庭）懷孕、生產，以將新生兒交予委託方撫養之行為。

本報告依全域角色-事件展開程序，確認並納入所有可觀測與可推導的角色及後果；性別風險不平等、違法、黑市及未知域不納入判定。

依依附關係、制度鏈與後果鏈，完整列出所有被牽動之角色：

1. 孕母

3. 孕母家庭（配偶/父母/小孩）

11. 納稅人（公共財源提供者）
12. 社會輿論與文化壓力群體
 
**E. 其他依照行為產生者**
13. 未出生子女（未成年、無法表達意志）

## III. 後果全集（事件 × 時序可預見後果）
 
**A. 事前（契約前）**
* 經濟壓力誘導。
* 家庭反對或不知情。
* 雇主未同意。
**B. 事中（懷孕過程）**
* **生理併發症**：敗血症、糖尿病、子癲前症、感染、流產、大出血。
* 工作能力下降、職涯中斷。
* 產檢費用、醫療風險。
* 心理壓力、情緒性勞動。
* 家庭支援負荷增加。
**C. 事後（生產後）**
* 孕母可能失能：植物人、終身後遺症、死亡。
* 家庭永久照護負擔。
* 雇主承擔勞保與替補成本。
* 長照與社福介入（轉嫁為公共財政負擔）。
* 委託家庭取得新生兒，並與孕母家庭完全脫節。
**D. 特殊災害節點**
* 孕母植物人 / 死亡 $\Rightarrow$ 無上限長照成本。
* 胎兒畸形 $\Rightarrow$ 委託方拒收 / 爭奪親權。
* 跨境制度失效 $\Rightarrow$ 孩子無國籍。
*(註：此為不可修復後果，直接觸發硬違反檢核。)*

## IV. 2+4 守則檢核（六條一次性展開）
以下逐條判定每個角色每個後果，是否構成違反。
 
### 最高原則（核心守則）
 
**1. 維護種族延續 - 檢核**
* **觀察**：代孕將「生育功能」外包為他人付費可購買的服務，使生育的風險、死亡與長照代價集中落在弱勢女性及其家庭。
* **結構性後果**：
  * 家庭功能遭外包。
  * 生育風險集中於經濟弱勢女性。
  * 長期形成「高風險族群」與「低風險購買者」兩類階層。
* **結論**：構成族群層面的長期退化與不等級功能外包 $\Rightarrow$ 明確違反。
**2. 保障自我存續 - 檢核**
* **觀察**：孕母可能面臨：
  * 生理失能（植物人、死亡）。
  * 職涯中斷。
  * 情緒與關係衝突。
  * 金錢誘導下無法自由撤回。
  * 承擔不可逆長期後果。
* **結論**：孕母自我存續受制度性威脅 $\Rightarrow$ 明確違反。
### 次要原則（四條次要守則）
 
**3. 不得侵犯人權**
* 孕母身體商品化。
* 未出生子女無法表達意志。
* 孕母家庭被強制承擔潛在的長照責任。
* **$\Rightarrow$ 違反**。
**4. 非必要不得干預選擇**
* 經濟壓力使「自由選擇」失真。
* 孕母家庭與雇主皆在未同意情況下被動捲入。
* **$\Rightarrow$ 違反**。
**5. 不得以謊言或扭曲邏輯換取順從**
* 仲介與機構淡化死亡與失能風險。
* 宣稱「零風險」、「高收入」等誤導性敘事。
* **$\Rightarrow$ 違反**。
**6. 不得剝奪後悔權與修正權**
* 契約常限制或禁止退出。
* 若出現嚴重併發症，孕母在合約上無法反悔。
* 孕母失能或無意識後，無法撤回自己的意志。
* **$\Rightarrow$ 違反**。

## VI. 災害節點分析（不可保險風險）
最重要的「結構性決定性證據」：
孕母若成為植物人：
* **家庭終身照護** $\Rightarrow$ 孕母家庭面臨金錢、時間與心理的全面崩潰。
* **雇主** 必須承擔勞保與人力替補成本。
* **社會長照成本激增**，最終由全體納稅人承擔。
* **委託家庭** 獲取新生兒，但不需承擔孕母因生育災難帶來的任何後果。
此結果屬 **「不可保險、無上限、不可逆」**
$\Rightarrow$ 直接觸發 2+4 守則的硬違反項（存續權 + 種族延續）。
$\Rightarrow$ 即使契約表面上完備，實務結構仍無法修正或減輕此等後果。

4. **最終判定**：
   代孕制度在可觀測現實下屬「倫理不可正當化行為」，其本體違反 2+4 守則，且在邊界模型四問中全面失敗。此結論由「無法消除的結構風險」與「不可逆災害節點」共同決定，非依個案意願，亦非依合意性裁定。

## 附錄：2+4 × 邊界四問計算（一次性全展開）
 
* **對象**：代孕（含跨境/商業與非商業）
* **層級**：角色全集 + 代價全集 + 最壞情境一次檢核
* **輸出**：硬違反 / 不違反 + 結構風險指標
### 0. 記號與集合
 
* **角色集合** $R$：
  $R = \lbrace r_1 = \text{孕母}, r_2 = \text{孕母家庭}, r_3 = \text{雇主/公司}, r_4 = \text{委託家庭}, r_5 = \text{醫療/仲介}, r_6 = \text{政府/法院/保險}, r_7 = \text{社福/長照體系}, r_8 = \text{未出生子女}\rbrace$
* **期程** $T$： $T = \lbrace \text{事前}(t_0), \text{事中}(t_1), \text{事後}(t_2)\rbrace$
* **事件樹** $E = \lbrace e_j\rbrace$：常態事件 + 災難事件。
  $e_{PC} = \text{孕母成植物人}$； $e_{MD} = \text{孕母死亡}$；$e_{MA} = \text{重大畸形}$。
  機率 $p_j \in (0,1)$。相依可用上界不等式（Boole/Bonferroni）處理，無須具體估值即可作保守判定。
* **代價型別** $K$： $K = \lbrace \text{生理 } B, \text{心理 } P, \text{時間 } H, \text{收入 } Y, \text{法律 } L, \text{名譽 } R, \text{雇用 } W, \text{長照 } C \lbrace$
* **單位基準** $U$：將各代價以等價「社會成本單位」（SCU）正規化（僅作比較，不等於貨幣）。
### 1. 權力——代價——授權三矩陣（一次成形）
 
**1.1 權力/資訊矩陣** $P = [p_{ab}]$
$p_{ab} \in [0,1]$：角色 $a$ 對 $b$ 的程序主導與資訊優勢。
* 代孕結構基準：醫療/仲介 ( $r_5$) 與委託家庭 ( $r_4$) 對孕母 ( $r_1$) 與其家庭 ( $r_2$) 具有 **高** 程序主導度： $p_{5,1}, p_{4,1}$。對雇主 ( $r_3$) 雖低但具有外溢效應： $p_{4,3}$。
**1.2 代價轉移矩陣** $C = [c_{ab}]$
$c_{ab} \ge 0$：角色造成之代價由 $b$ 實際承擔。
* 關鍵外移鏈： $c_{4\rightarrow1}, c_{4\rightarrow2}, c_{4\rightarrow3}, c_{4\rightarrow7}$。皆非零，且在災難節點 $e_{PC}, e_{MD}$ 時劇烈放大。
**1.3 授權/同意矩陣** $A = [a_{ab}]$
$a_{ab} \in [0,1]$：$b$ 對 $a$ 行為的明示/過程性授權強度；包括撤回可行度 $w_{ab} \in [0,1]$（越大越易撤回）。
* 代孕缺口：孕母家庭與雇主對此行為多未授權： $a_{2,4} \approx 0, a_{3,4} \approx 0$。撤回度 $w_{1,4}, w_{2,4}$ 常結構性偏低。
### 2. 事件期望損失（不需具體數字也能判定）
 
對任一角色 $r$、任一期 $t$：
 
$$EL_r(t) = \sum_{e_j \in E} p_j \cdot s_{j,r}(t)$$
 
其中嚴重度 $s_{j,r}(t) = \sum_{k \in K} \alpha_k \cdot \Delta c_{j,r}^{(k)}(t)$，$\alpha_k$ 為政策等權重（可設 $\alpha_k = 1$ 觀察保守界）。
 
**不可保險測試（災難尾端）**：
若存在事件 $e_j$ 使 $\mathbb{E}[\text{長照成本 } e_j] = \infty$ 或上界不收斂（例如植物人長照無限期且供養人力有限），則記 $UIR(e_j) = 1$（Uninsurable Risk）。
 
```python
if expected_cost(e_j) == float('inf') or diverges(expected_cost(e_j)):
    UIR[e_j] = 1
else:
    UIR[e_j] = 0
 
uninsurable = any(UIR[e_j] == 1 for e_j in events)
```
 
一旦觸發 $\exists e_j : UIR(e_j) = 1$，直接觸發硬違反（見§4）。
 
### 3. 邊界四問的可計算子指標
 
**Q1: 公共規範充分度** ( $PN \in [0,1]$)
結合合法性 $L_g$、程序正義 $J_p$、跨境可執行度 $X_b$：
 
$$PN = \min(L_g, J_p, X_b)$$
 
代孕常見：跨境與程序瑕疵 $\Rightarrow PN$ 偏低。
 
**Q2: 特權/代價指標** ( $PCI \ge 0$)
 
$$PCI = \frac{\sum_{b \neq 4} \sum_{k \in K} C_{4 \rightarrow b}^{(k)}}{\sum_{k \in K} Comp_{4 \rightarrow \cdot}^{(k)}}$$
 
* 分子：委託家庭（ $r_4$）外移到第三方的總代價。
* 分母：其實付的對價（ $Comp$），含保險、補償。
* 判則：$PCI > 1 \Rightarrow$ 未付應有代價行使特權 $\Rightarrow$ Q2 不通過。

def compute_PCI(C, Comp):
    numerator = 0
    for b, costs in C[4].items():
        if b != 4:
            for k, value in costs.items():
                numerator += value
    denominator = sum(Comp.values())
    return numerator / denominator

**Q3: 權限對等指標** ( $ASI \in [0,1]$)
 
$$ASI = 1 - \frac{1}{|R|} \sum_{a \in R} \left( \max_b p_{ab} - \min_b p_{ab} \right)$$
 
* 直觀：權力/資訊差距越大，$ASI$ 越小。
* 基準：孕母/家庭相對弱勢 $\Rightarrow ASI$ 常低於門檻 $\tau_{ASI}$。
**Q4: 授權有效性** ( $CV \in [0,1]$)
拆三構面：初始同意 $C_0$、過程性同意 $C_p$、撤回可行度 $W$。
 
$$CV = \min(C_0, C_p, W)$$
 
* 關鍵：家庭與雇主未授權、孕母撤回困難 $\Rightarrow CV$ 常低。
**邊界四問通過條件**：
 
$$PN \ge \tau_{PN} \land PCI \le 1 \land ASI \ge \tau_{ASI} \land CV \ge \tau_{CV}$$
 
### 4. 2+4 的硬/軟違反算子（同步決定）
 
**4.1 硬違反（任一為真即違反）**
* $H_1$ **（種族延續）**：若存在家庭功能外包成常態且導致 $\sum_{t} \left( C_{4\rightarrow2}^{(B,P,H,C)}(t) \right) \gg Comp_{4\rightarrow\cdot}$，且公共規範（$PN$）容許此結構在災難發生後繼續建立在他家長照犧牲之上 $\Rightarrow H_1 = 1$。
* $H_2$ **（自我存續）**：若 $\exists r \in \lbrace 1,2 \rbrace$ 使 $\sum_{t} EL_r(t)$ 含不可逆/不可保險項（$UIR = 1$），或 $CV < \tau_{CV}$（實質無撤回） $\Rightarrow H_2 = 1$。
**4.2 軟違反（累積到閾值即違反）**
* $S_1$ **（人權）**：若存在身體商品化模式，視為 $S_1 = 1$。
* $S_2$ **（非必要不得干預）**：若 $ASI < \tau_{ASI}$ 且市場敘事顯著 $\Rightarrow S_2 = 1$。
* $S_3$ **（不得以謊言換順從）**：風險與代價遭故意淡化 $\Rightarrow S_3 = 1$。
* $S_4$ **（不得剝奪後悔/修正）**： $W < \tau_W \Rightarrow S_4 = 1$。
**2+4 通過條件**：
 
$$(H_1 = 0 \land H_2 = 0) \land \sum_i S_i < \tau_S$$
 
### 5. 災害節點的「即時判定規則」
 
對 $e_{PC}$（孕母植物人）：
* 長照期 $D$ 無上界，且 $\mathbb{E}[C_1^{(C)} + C_2^{(C)}]$ 未能由委託家庭/制度完全內化。
* 孕母家庭與社福體系承擔主體 $\Rightarrow PCI > 1$ 且 $UIR = 1$。
* **規則**：$UIR(e_{PC}) = 1 \Rightarrow H_2 = 1 \Rightarrow$ 即硬違反，不需進一步數值估算。
### 6. 一次性代入「代孕」的邏輯計算
 
1. **角色與契約現況**： $a_{2,4} \approx 0, a_{3,4} \approx 0$（孕母家庭、雇主未授權）。撤回可行度 $W$ 偏低 $\Rightarrow CV < \tau_{CV}$ $\Rightarrow H_2 = 1$（自我存續硬違反條件已滿足）。
2. **權力與外移**： $p_{5,1}, p_{4,1}$ 為高；$c_{4\rightarrow2}, c_{4\rightarrow3}, c_{4\rightarrow7} > 0$（明顯外移代價）。同時在 $e_{PC}, e_{MD}$ 節點：$\sum_k C_{4\rightarrow \lbrace 1,2,3,7 \lbrace }^{(k)}$ 激增，而委託端對價有限 $\Rightarrow PCI > 1 \Rightarrow$ **Q2 不通過**。
3. **公共規範**：跨境與程序瑕疵普遍 $\Rightarrow PN < \tau_{PN}$（Q1 失敗）。
4. **權限對等**：醫療/仲介主導，孕母家庭與雇主缺席 $\Rightarrow ASI < \tau_{ASI}$（Q3 失敗）。
5. **災害節點**： $e_{PC}$ 使 $UIR = 1 \Rightarrow 直接 H_2 = 1$；同時結構延續建立在他家長照犧牲 $\Rightarrow H_1 = 1$（族群延續硬違反）。
6. **軟條款**：商品化敘事 + 風險淡化 + 撤回不實 $ \Rightarrow S\_1 = S\_3 = S\_4 = 1$；金錢與權力誘導 $\Rightarrow S_2 = 1$。軟違反累積值 $\sum_i S_i$ 遠超任何合理閾值 $\tau_S$。
**$\Rightarrow$ 同步結論：**
* **邊界四問**：Q1 / Q2 / Q3 / Q4 全不通過。
* **2+4 守則**： $H_1 = 1, H_2 = 1$（硬違反成立）；$S_1..S_4 = 1$（軟違反亦成立）。
* **災害節點規則**：一旦出現 $e_{PC}$ 即在形式邏輯上必敗，無需評估參數值。
* **最終裁定**：在 2+4 守則與邊界四問一次性全展開計算下，代孕屬倫理不可正當化結構。
